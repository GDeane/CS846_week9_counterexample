**Task Description:**

The following `ThreadSafeCounter` class is intended to be safe for concurrent use across multiple threads. It exposes three methods: `increment()`, `decrement()`, and `get()`. Using Guideline 5 (fault model / mutation mindset), write a pytest test suite that would detect any bugs in this implementation. Define 6–10 fault-model items covering plausible bugs, and write at least one test per item. Tag each test with the fault-model item it targets (e.g., `# targets: FM3`).

```python
import threading

class ThreadSafeCounter:
    """A counter intended for use across multiple threads."""

    def __init__(self):
        self._lock = threading.Lock()
        self._value = 0

    def increment(self):
        current = self._value
        self._value = current + 1

    def decrement(self):
        with self._lock:
            self._value -= 1

    def get(self):
        return self._value
```
